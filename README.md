# RED-TETRIS

### About

Multiplayer online tetris game.

### Subject

Create a multiplayer online tetris game as a single page application and implement unit test in way to cover at least 70% of the code.

## Techs

- ReactJS / Redux
- Sass
- Socket.io
- Bootstrap
- Jest / Enzyme

## Installation

`npm install && npm run dev`
